Informe del Proyecto

BrainTumor Detector - Sistema de Deteccion de Tumores Cerebrales en MRI usando CNN

Resumen

Sistema completo de clasificacion de imagenes MRI cerebrales en 4 clases: glioma, meningioma, notumor, pituitary. El proyecto incluye pipeline de ML (EDA, preprocesamiento, CNN propia, Transfer Learning), API REST para inferencia, frontend interactivo y opciones de despliegue (Docker y systemd).

Componentes del Sistema

ML Pipeline

  • 5 notebooks en notebooks/ (EDA, preprocesamiento, CNN, Transfer Learning, Evaluacion)
  • 1 notebook adicional: 06_visualizaciones_mejoradas.ipynb
  • Modelos: models/cnn_custom.keras, models/resnet50_finetuned.keras/

API

  • POST /predict en Flask (multipart/form-data, campo image)
  • Respuesta: {"tipo_tumor", "probabilidad", "probabilidades"}
  • Puerto: 5000
  • Archivos: api/app.py, api/inference.py

Frontend

  • frontend/index.html - Version original
  • frontend/profesional.html - Version mejorada con imagenes decorativas, radar chart y diseno profesional
  • Drag & drop, preview con animacion de escaneo, resultados con codigo de colores

Despliegue

  • Dockerfile - Imagen Linux con Python 3.11 + TensorFlow + gunicorn
  • braintumor-api.service - Servicio systemd con reinicio automatico
  • Documentacion en docs/despliegue.md

Instrucciones de Uso

API local

cd api
pip install -r requirements.txt
python app.py
# En otro terminal:
curl -X POST -F "image=@ruta/imagen.jpg" http://localhost:5000/predict

Docker

docker build -t braintumor-api .
docker run -d --name braintumor-api -p 5000:5000 braintumor-api
curl -X POST -F "image=@ruta/imagen.jpg" http://localhost:5000/predict

Frontend

Abrir frontend/profesional.html en el navegador
(La API debe estar corriendo en localhost:5000)
